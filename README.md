# SmartMonument 

Chatbot IA de détection et d'exploration des monuments historiques du Maroc.

## Description
Application web qui identifie automatiquement un monument marocain 
à partir d'une photo et permet de poser des questions via un chatbot IA.

## Technologies
- Python
- TensorFlow / Keras (MobileNetV2)
- Streamlit
- Ollama + Llama3

## Fonctionnalités
- Détection de 21 monuments marocains
- Accuracy : 75.85%
- Fiche patrimoniale
- Chatbot RihlaBot

## Installation
pip install -r requirements.txt
streamlit run app.py
